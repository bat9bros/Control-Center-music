# Control Center Music

Offline launcher and track manager.

![Launcher](https://github.com/user-attachments/assets/d419a661-4806-406b-98bf-0b92cfe4b8c9)

## Features

- Launch any application
- Manage tracks
- Drag & drop track statuses
- Mini mode (Ctrl + Shift + B)
- Local storage
- Fully offline

## Installation

npm install

## Run

npm start
